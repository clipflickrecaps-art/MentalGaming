PHASE 3: POST ORDER REVIEW SYSTEM
==================================================

Implement automatic review flow after order completion.

When admin completes an order:
- Send order completed message to user
- Add inline buttons:
  - ⭐ Leave Review
  - ❌ Skip Review

Do not use normal reply keyboard for this review flow.
Use inline buttons.

If user clicks Skip Review:
- Return user to main menu
- Mark review as skipped

If user clicks Leave Review:
1. Ask user to select star rating from 1 to 5
2. Ask user to write review message
3. Save review
4. Auto post review to configured review channel

Review channel message must include:
- Star rating
- Review message
- Product name
- Product preview/image if available
- Order ID
- Customer display name
- Customer username
- Paid amount
- Completed time
- Review submitted time

Admin settings:
- Review channel ID
- Enable/disable review system
- Optional MC reward for review

Rules:
- One review per completed order
- User cannot review cancelled/refunded orders
- User cannot submit duplicate review for same order

PHASE 5: TIER SYSTEM
==================================================

Implement Dual Tier System:
- Lifetime Tier
- Active Tier

Lifetime Tier:
- Based on lifetime total completed paid orders
- Never decreases
- Used as achievement/history

Active Tier:
- Based on yearly or last 365 days completed paid orders
- Used for current benefits
- Can decrease if yearly spending requirement is not maintained

Tier thresholds:

Bronze:
- 0 KS to 500,000 KS

Silver:
- 500,000 KS to 2,000,000 KS

Gold:
- 2,000,000 KS to 6,000,000 KS

Platinum:
- 6,000,000 KS to 10,000,000 KS

Diamond:
- 10,000,000 KS+

Tier MC bonus rates:

Bronze:
- 0.1% MC bonus

Silver:
- 0.3% MC bonus

Gold:
- 0.5% MC bonus

Platinum:
- 1% MC bonus

Diamond:
- 2% MC bonus

Important:
- Tier benefits are cumulative.
- Higher tiers inherit all lower-tier privileges/features.
- Percentage-based MC bonus must NOT stack.
- Only the highest active tier MC bonus rate applies.

Example:
Diamond user gets Diamond MC bonus rate only: 2%.
Do not calculate 0.1 + 0.3 + 0.5 + 1 + 2.

Tier benefits:

Bronze:
- 0.1% MC bonus

Silver:
- Bronze benefits
- 0.3% MC bonus

Gold:
- Silver benefits
- 0.5% MC bonus
- Priority support

Platinum:
- Gold benefits
- 1% MC bonus
- Exclusive flash sales

Diamond:
- Platinum benefits
- 2% MC bonus
- Birthday MC bonus
- Special coupons
- Early access promotions

Mini App Profile page must show:
- Lifetime Tier
- Active Tier
- Lifetime spend
- Yearly / 365-day spend
- Progress to next tier
- Current MC bonus rate
- Current benefits

Mini App:

- Show MC history

Admin:

- View MC logs
