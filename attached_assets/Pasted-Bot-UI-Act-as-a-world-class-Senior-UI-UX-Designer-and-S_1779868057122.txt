Telegram Bot Integration:
The Mini App must work exactly with Telegram Bot behavior.

Telegram Bot Commands:
- /start
- /shop
- /wallet
- /orders
- /topup
- /support
- /profile
- /admin

Bot Behavior:
- /start sends welcome message with “Open Store” WebApp button
- /shop opens Mini App shop page
- /wallet opens wallet page
- /orders opens order history page
- /topup opens top-up page
- /support opens support page
- Bot sends notification when:
  - Order created
  - Order processing
  - Order success
  - Order cancelled
  - Top-up approved
  - Top-up rejected
  - Admin sends message
  - Flash sale starts
