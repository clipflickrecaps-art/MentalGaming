PAYMENT METHOD & DIRECT PAYMENT SYSTEM
==================================================

Mental Coins must NOT be used as a normal checkout payment method for regular products.

Mental Coins are only for:
- Redeem products
- Discount coupons
- Promotions
- Spin rewards
- Review campaign rewards

Normal product checkout payment methods:

- Wallet Balance
- Direct Payment

Display label:

English:
- Direct Payment

Myanmar:
- တိုက်ရိုက်ငွေပေးချေမည်

Admin must be able to configure payment accounts from the admin panel.

Supported account types:

- KPay
- Wave Pay
- AyaPay
- Bank Account
- Custom Payment Method

Direct Payment Flow:

1. User selects Direct Payment.
2. Show available payment accounts configured by admin.
3. User transfers money.
4. User must upload payment proof screenshot.
5. Transaction ID field is optional.
6. Create order with status:
   - Payment Pending
7. Admin reviews payment proof.

Admin actions:

- Confirm Payment
- Reject Payment

If payment is rejected:
- Order cannot continue to delivery.
- Order status becomes Payment Rejected.

If payment is confirmed:
- Order status becomes Payment Confirmed.
- Order moves to Delivery Pending.

Order Statuses:

- Payment Pending
- Payment Confirmed
- Delivery Pending
- Completed
- Payment Rejected
- Delivery Rejected
- Refunded to Wallet

Delivery Failure Handling:

If payment has already been confirmed but delivery cannot be completed because of:
- Wrong customer information
- Wrong Game ID
- Wrong Server ID
- Product unavailable
- Stock unavailable
- Delivery failure
- Any admin-approved reason

Admin must be able to:

- Reject Delivery
- Refund to Wallet

Example:

User pays 12,000 KS using Direct Payment.
Admin confirms payment.
Delivery later fails.
Admin selects Refund to Wallet.

System automatically:
- Adds 12,000 KS to the user's wallet balance
- Creates refund history
- Updates order status to Refunded to Wallet
- Saves all actions in order history

Product Payment Options:

- Wallet Only
- Direct Payment Only
- Wallet + Direct Payment
- MC Redeem Only
- Coupon Eligible

Bot and Mini App must use the same payment rules, payment statuses, refund logic, order statuses, and checkout behavior.
